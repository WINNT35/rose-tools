v0.1 — 2026-04-28

Initial release of Rose Tools. This release establishes the core architecture for a native Windows port of VMware guest tools built on the open-vm-tools protocol stack, and marks the first entry in the Rose Tools documentation.

Core

The backdoor communication layer, message channel, and RPCI sender are taken directly from open-vm-tools and compiled for Win32 using MinGW. Original shims are provided where full ports are pending. A clean plugin architecture is introduced as a GLib-free equivalent of the upstream design, written in C89.

Outer Ring

The outer ring consists of two components; checkvm.exe, a standalone VM detection utility, and vmrosd.exe, a console daemon that manages the service lifecycle and loads plugins. A single plugin, guestInfo, reports the guest OS name, hostname, and primary IP address to the host.

Confirmed Working

All three v0.1 goals are confirmed working on Windows XP (NT 5.1) SP3 under VMware Workstation; VM detection, persistent heartbeat, and guest information visible to the host.
